Theme Features

  • Next.ts   :   Theme is completely converted in Next.ts so it is easy to use server side rendering and it also provides better SEO
  • GraphQL Supported   :   We have integrated graphQL here to make it easier to integrate your API.
  • Responsive Design   :   Looks great on desktops, laptops, tablets and mobiles. Give it a try on your iPhone or iPad now.
  • RTL Supported   :   All inner pages of Bigdeal are RTL supported.
  • Made with Sass   :   Bigdeal is made with sass.
  • W3 Validator   :   This theme coded with beautiful and clean codes! Some powerful HTML files 100% valid W3 web standards.
  • Bootstrap 5   :   Bigdeal is made with latest version of Bootstrap.
  • Google Font   :   You can use unlimited Google fonts.
  • Product Hover Variation   :   This theme has 4 product hover variations
  • React Slick Carousel   :   We have used it for sliders.
  • React Toastify   :   We have used it for Notification.
  • Add to Cart   :  We have implemented Add to cart functionality, where user can store multiple products in cart to checkout all at once. .
  • Quick-view   :   Every product has Quick view modal popup.
  • Wishlist   :   Click on wishlist button to add the product to your wishlist.
  • Product Compare   :   Click on Compare button product to add it in the compare list. Maximum 4 product are allowed to be compared.
  • Product Details   :  Every Product have individual details page and 10+ different product pages, so that you are never out of options.
  • One Page Checkout   :  We have provided a single checkout page to reduce the process of checking out,and make it easier for the customer to buy things faster.
  • Shop Page   :  We have added 3 different shop pages. Choose any that you prefer for your site.
  • Filter   :  3 Different type of custom filters have been provided (tag, color, size) it's customizable, you can customize it according to your logic.
  • Payment Method   :  We have added paypal as a payment method.
  • Blog Page   :  There are three different variations for blog pages.
  • Inner Page   :  Bigdeal theme includes 30+ inner pages.

Start Building

Prerequisite
Install Node.js and NPM : Download Node.js from https://nodejs.org/en/download/ and install it on your system. Use latest version of node i.e. 16.15.0. NPM comes bundled with Node.js
Theme

When you extract file from download you will find following folder:

  • Theme
  • API
  • EmailTemplate
  • Documentation
To Start working on running your frontend you need to follow below step:
  • Extract API folder as well as Theme folder
  • Then run Api project first by doing:
    • npm i
    • npm start

      • now your Api port should be running. You can check it out with url which is provided in the terminal

    Note: For getting data in frontend (in your all demos) it is necessary that you run this api first.
  • Now it's time to run your frontend to see theme through which your dreams will come true
    • npm i
    • npm run dev
    • Once your frontend runs successfully you can access it with this link http://localhost:3000

SCSS

  • All Theme design related SCSS can be found in this folder /public/assets/scss/theme
    • _style.scss (Complete theme scss you can find in this file)
    • _responsive.scss (Responsive related styles of complete theme can be found here)
    • _variable.scss (You can find all Common variables of complete theme in this file)
    • _dark.scss (for Dark mode of theme)
    • _rtl.scss (scss of complete RTL theme can be found here)
    • _menu.scss (for menu of theme)
  • app.scss //Import statements of all scss will be here
  • color.scss // all colors are define here

Unlimited Color

    • Yes, You read correct!! we are here with unlimited color option. To change the color of complete theme you need do the following:
    Go to color.scss in /public/assets/scss/color.scss and change your default color of config file to whatever you want in --theme-color1 and --theme-color2. that's it you can have a theme with a color of your choise. Suppose you want color-1 as default color, you can set your fav color in color-1 like this:

    • html[class="color-1"] {
        --theme-color1: #00baf2;
        --theme-color2: #1c3481;
    }

    Removal of demos

    It's bit easy to remove demos from theme

    Suppose if you don't want other demos except demo which you like then just follow below steps:

    • Remove all layouts from pages >> Layouts except the one which you like
    • Remove all other layout from views >> layouts except the one you imported in your demo
    • Remove other product box from views >> layouts >> widgets >> product-box except the one which you imported in your upper component
    • Also you can remove all extra pages as per your requirement

    Removal/ Change Loader

    Suppose if you want to remove /change the loader which you can see in page load that you can change it by:
    • Going to bigdeal-next-ts/common/Loader.js where you can find code for loader, you can just remove that to remove a loader.
    • Now suppose if you want to change design of that loader you need to take that class and find its scss and change it as per your requirements
